安卓开发环境的搭建

  • Android

按照视频搭建安卓开发环境，过程中还是出现了很多问题。

安卓开发环境详细的安装过程可以看五步搞定Android开发环境部署——非常详细的Android开发环境搭建教程这篇博客，这里介绍的是独立安装工具的。
我下载的是集成环境的，少去了需要自己安装SDK和ADT工具的步骤。

将下载的文件解压后

打开eclipse通过Android SDK Manage添加新的软件包时却看不见可以更新的安装包。

点击更新安装已有的软件包时显示连接不上谷歌的网址，因为国内有墙，所以会出现获取失败的情况。

最后是以下面的方法解决了这个问题，打开Android SDK Maneger->Tools修改图中的选项。

下面才是Android SDK Maneger正确的打开方式。

然后便可以勾选要更新的软件包啦，出现有些更新失败的情况只要不断重新关闭SDK Maneger再开启更新就可以了。

解决了SDK问题又遇到ADT版本太低的问题，由于更新了SDK而ADT的版本还没跟上，设置Android Preferences的时候就会出现This Android SDK requires Android Developer Toolkit version 23.0.0 or above…….
解决方法如下：找到android-sdk-windows\tools\lib下的plugin.prop文件修改其中的版本号与所安装的ADT版本号相同就可以了。
